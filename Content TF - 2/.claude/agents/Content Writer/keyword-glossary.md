# Keyword Glossary

Quick reference for prohibited terms and approved alternatives. Check words against this list before finalizing content.

> **Client setup**: The sections below are universal writing standards that apply to most clients. Replace the **Regulatory & Scientific Terms** section with the client's own industry-specific compliance terms.

---

## Absolutes - NEVER USE

| Avoid | Use instead |
|-------|-------------|
| All | Many, Most, Significant |
| Always | Regularly, Typically, Commonly, Usually, Mostly |
| Any | Many, Most, Significant |
| At every step | [Remove or rephrase without absolute] |
| Assure | [Cannot promise - remove] |
| Eliminate(s) | Minimizes, Reduces, Helps reduce |
| Ensure(s) | Help ensure, Enable, Support, Allow, Facilitate, Permit |
| Essential | Important |
| Every time | Regularly, Typically, Commonly, Usually, Mostly |
| Everyone | Many, Most, Significant |
| Guarantee(s), Guaranteed | Designed to, Anticipate, Help, Enable, Support, Allow, Facilitate, Permit |
| 100% accuracy | Accurate |
| No contamination | Minimizes contamination |
| Obtain/obtaining | Enable, Enabling |
| Remove(s) | Minimizes, Reduces, Helps reduce |

## Superlatives - AVOID (require data to substantiate)

| Avoid | Use instead |
|-------|-------------|
| Best, Best-in-class | Among the best, Outstanding, Superior |
| Breakthrough | Innovative, Advanced |
| Cutting Edge | Innovative, Advanced |
| Effortless, Effortlessly | Easy, Easily |
| Fastest | Among the fastest, Quick, Outstanding |
| Gold standard | Trusted, Widely used (requires legal approval) |

| Perfect | Excellent, Well suited for, Outstanding, Superb |
| Revolutionary | Exceptional, One of... |
| Seamless, Seamlessly | Easy, Easily |
| State-of-the-Art | Innovative, Advanced |
| Unique | Exceptional, One of... |
| Unmatched | Exceptional, Excellent, Remarkable, Well suited |
| Unparalleled | Exceptional, Excellent, Remarkable, Well suited |
| Unprecedented | Exceptional, Excellent, Remarkable, Well suited |
| Unrivaled | Exceptional, Excellent, Remarkable, Well suited |

## Antitrust/Competitive Terms - NEVER USE

| Avoid | Use instead | Reason |
|-------|-------------|---------|
| Dominate/dominance | Succeed, Achieve goals/targets, Significant position | Antitrust risk |
| Eliminate/reduce competition | Compete vigorously, Challenge competitors, Effectively compete, Benefiting customers | Antitrust risk |
| Market | Industry, Segment, Business, [specific product area] | Antitrust risk + vague |
| Market leader | Large player | Antitrust risk |
| Market shares | Sales, Volume | Antitrust risk |
| Price increases/raise prices | Reduce costs, Generate efficiencies, Increase/expand sales | Antitrust risk |

## Claims Requiring Data/Evidence - USE CAREFULLY

| Term | Requirement |
|------|-------------|
| First | Only with documented evidence; use "One of the first" or "Our first" |
| Industry leader | Requires legal approval + narrow tailoring |
| Proven | Requires written proof; otherwise use "Widely used" |
| Preferred | Requires supporting data; use "Widely used" |
| Save money | Requires data on file; use "Efficient" |
| Save time | Requires data on file; use "Efficient" |

## "Standard" Claims - SOFTEN

| Avoid | Use instead |
|-------|-------------|
| Global standard | Widely adopted, Widely used, Well known |
| Industry standard | Widely adopted, Widely used, Well known |
| Laboratory standard | Widely adopted, Widely used, Well known |

## Quantitative Extremes - AVOID

| Avoid | Use instead |
|-------|-------------|
| Infinite | Exceptional, Many, Significant, Vast |
| Longest | Up to [number], As little as [number] |
| Most | Widely used, Trusted |
| Shortest | Up to [number], As little as [number] |
| Sole | Exceptional, One of... |
| Only | Exceptional, One of... |
| Only commercially available | Our first, One of the first, One of the few commercially available |
| Unlimited | Exceptional, Many, Significant, Vast |

---

## Action Verbs - SOFTEN PROMISES

| Avoid | Use instead |
|-------|-------------|
| Deliver | Help deliver, Enable, Support, Allow, Facilitate, Permit |
| Provide | Help provide, Enable, Support, Allow, Facilitate, Permit |

---

## Vague/Ambiguous Terms

| Avoid | Use instead | Reason |
|-------|-------------|---------|
| Early access | Access | Implies pre-approval access |
| Expert | Experienced professional, Professional, Specialist, Knowledgeable | Vague |
| Expertise | Experience | Vague, implies unqualified promise |
| New | [Use within 6 months of launch only] | Product ages with material |
| Novel | Exceptional, One of... | Difficult to prove |
| Partner/Partnered | Collaborator, Collaborated, Worked with | Creates agency relationship legally |
| Patented technology | Proprietary technology | Not all IP has patent protection |

---

## Product Descriptors - QUALIFY

| Avoid | Use instead |
|-------|-------------|
| Dependable/Convenient (absolute) | Designed to be dependable/convenient |
